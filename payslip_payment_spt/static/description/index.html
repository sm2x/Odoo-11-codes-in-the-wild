Odoo Payslip Payment Register (Community Only)

This module is aim to register the payslip payments for payrolls.
  • Features
  • Screenshots

Primary Features

  • Easy interface to manage odoo payroll payment.

  • Allow to register payment in payslip.

  • Allow to cancel paid payslip.

  • Mass Payslip batch payment register.

  • Mass Payslip batch cancel.
