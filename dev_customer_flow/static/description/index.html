Customer/Supplier Approval Workflow

DevIntelle Consulting Service Pvt Ltd

Overview

Odoo Customer/Supplier Approval Workflow Process will help to manage customer and supplier confirm and validation process by work flow by specific user approvals. it will generate uniq number for customer and supplier approval to easy identification.

Feature

Customer/Supplier Approval Workflow validation Process When any new customer or supplier created in odoo
Only access rights user can validate newly Customer/Supplier work flow .
Generate unique number for customer and supplier approval to easy identification.
Approval user can approve or confirm Customer/Supplier work flow .
Easy view to see customer/supplier approval status on kanban view .
Quick filter to check status of customer/supplier approval.
Only approval customer will be listed on sale order, delivery order, customer invoice into customer dropdown
Only approval supplier will be listed on purchase order, shipment, vendor bill into vendor dropdown
Only approval vendor will be listed product supplier list

How To Work

Access rights for user can validate newly Customer/Supplier work flow.
Only allowed user can validate newly Customer/Supplier work flow
Access rights for user can approve validated Customer/Supplier work flow
Generate unique number for customer and supplier approval to easy identification.
Easy view to see customer/supplier approval status on kanban view & Quick filter to check status
Only approval customer will be listed on sale order screen into customer dropdown
Only approval supplier will be listed on purchase order into vendor dropdown.
Only approval customer will be listed on delivery order screen into customer dropdown
Only approval customer will be listed on customer invoice screen into customer dropdown.
Only approval customer will be listed on Customer Voucher screen into customer dropdown.
Only approval supplier will be listed on Vendor receipt into vendor dropdown .
Only approval supplier will be listed on Product Supplier into vendor dropdown
